# Tugas-md5-pytho# Program Cek Perubahan Data Profil User

Program Python sederhana menggunakan algoritma hash MD5 untuk mendeteksi perubahan data profil user.

## Fitur
- Input nama, email, dan nomor HP
- Membuat hash MD5
- Membandingkan data lama dan data baru
- Menampilkan status perubahan data
